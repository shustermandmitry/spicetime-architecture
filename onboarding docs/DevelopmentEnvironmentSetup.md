# Set Up the Development Environment

Follow these steps to set up your development environment:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd <repository-name>
   ```

2. Run the setup script:
   ```bash
   ./setup.sh
   ```

3. Start the development server:
   ```bash
   pnpm run dev
   ```
